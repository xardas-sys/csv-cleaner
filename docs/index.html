How the work stays controlled

Rules first. Changes second.

  1. 01

    Preserve

    The original file remains untouched. Work happens on a separate copy.

  2. 02

    Agree

    Record keys, duplicate retention, formats, and acceptance checks are explicit.

  3. 03

    Transform

    Deterministic Python applies only the agreed cleaning rules.

  4. 04

    Reconcile

    Input rows, output rows, removals, and hashes are reported together.
